📌신규아이템 : 유령의귀걸이가 추가하였습니다.

　

　

📌대장군목걸이의 옵션을 상향하였습니다.

　

📌모든 아이템의 옵션을 하향조절하였습니다.

( 대장군목걸이 제외 )

　

📌파티장이 파티탈퇴시 파티가 해산 되도록 변경하였습니다.

　

📌가넷상점에 10위랭킹진입 물약을 추가하였습니다.

　

📌더이상 포노스 오크 변신이 되지 않도록 변경하였습니다.

　

📌Lv 86 / 88 지배 변신을 삭제하였습니다.

　

📌변신 지배 반지 사용시 능력치를 상향하였습니다.

( pvp추타/리덕 +5 / Hp +300 / 모든스텟 +2 / 랭킹변신은 모든스텟 + 3 나머지는 동일 )

　

📌로테이션 감시자리퍼→준보스 사냥시 기운잃은 유령귀걸이 / 대장군 목걸이가 드랍되도록 추가하였습니다.

　

📌로테이션 / 발록 / 기르타스 보스 사냥시 유령귀걸이가 드랍되도록 추가하였습니다.

　

📌준보스 / 로테이션보스 가넷 드랍률을 상향하였습니다.

　

📌보스몬스터의 데미지를 하향하였습니다.

　

📌킬데스초기화 / 클래스변경 가격을 상향하였습니다.

　

📌진 데스나이트 세트가 기운잃은 → 완제 판매로 변경하였습니다.

　

📌진데스보물상자에서 축복부여주문서를 삭제하였습니다.

　

📌진데스보물상자에서 데스주문서를 추가하였습니다.

( 일정확률로 획득가능하며 사용시 킬은유지 1데스만 사라집니다. )

　

📌진데스보물상자에 메티스의축복주문서를 추가하였습니다.

　

📌진데스보물상자에 흑사코인을 추가하였습니다.
